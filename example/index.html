PUSH VALUE TO URL PARAMETERS

Code Documentation

Initialize

var pushToUrl = new pushToUrl();

Add key and value to url params

pushToUrl.add({key: 'name', value: 'John'});

Get selected key value from url params

pushToUrl.get('name');

Delete selected key from url params

pushToUrl.remove('name');

Remove all url params

pushToUrl.removeAll();
